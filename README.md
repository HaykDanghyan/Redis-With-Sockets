# Redis database implementation
```bash
make Redis & ./Redis
```

## Open another terminal

```bash
make client && ./client
```

## Main Functionality 

- SET
- GET
- LPUSH 
- RPUSH
- LRANGE
- HGET
- HSET 
- LLEN

## For using json, install nlohmann-json3
```bash
sudo apt install nlohmann-json3-dev
```
